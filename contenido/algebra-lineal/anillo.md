## Anillo

Sea $A$ un conjunto y sean $+$ y $*$ operaciones.

Entonces $(A, +, *)$ es un _anillo_ si:

1. $(A, +)$ es un grupo abeliano.
2. $*$ es asociativa y tiene elemento neutro.
3. Se cumplen las propiedades distributivas, es decir:
$$a * ( b + c ) = a * b + a * c$$
$$(b + c) * a = b * a + c * a$$
